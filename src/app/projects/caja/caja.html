Detección de Plagas

Este proyecto es desarrollado en conjunto con tres estudiantes de electrónica, quienes deben diseñar un mecanismo de detección de chizas en cultivos de papa mediante el análisis de frecuencias de audio, complementado con una cámara que permite identificar el estado de las papas.

Avances del proyecto

  • Plantado las papas de control
  • Trabajo con la cámara y el Raspberry Pi para la detección de plagas
  • Avance en el crecimiento de las papas

Próximamente

  • Instalar el micrófono y giroscopio
  • Toma de datos
  • Análisis de datos
